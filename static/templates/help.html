Comprehensive assistance for using the VisoLearn platform

Getting Started

Welcome to VisoLearn, an adaptive image education platform designed to support learning in children with autism spectrum disorder. This guide will help you navigate the platform and make the most of its features.

Basic Navigation

  • The main interface is divided into sections for generating images, interacting with the chatbot, and tracking progress.
  • Use the controls in the left panel to configure settings for the child you're working with.
  • The center area displays the generated image and conversation history.
  • The right panel shows progress tracking and details that need to be identified.

Generating Educational Images

  1. Enter the child's information:
    • Age: Enter the numerical age of the child
    • Autism Level: Select Level 1 (requiring support), Level 2 (requiring substantial support), or Level 3 (requiring very substantial support)
  2. Choose a topic focus: Enter a specific subject area you want to target (e.g., "animals", "emotions", "daily routines")
  3. Add treatment plan details: Optionally include specific goals or approaches from the child's treatment plan
  4. Customize settings:
    • Allowed Attempts: How many tries the child gets before advancing to a new image
    • Details Threshold: Percentage of details needed to identify before advancing difficulty
    • Image Style: Choose from Realistic, Illustration, Cartoon, Watercolor, or 3D Rendering
  5. Click "Generate Image" to create a customized educational image

Tip: Choosing the Right Difficulty

The system starts at "Very Simple" difficulty and will automatically progress as the child succeeds. However, you can generate a new image at the current difficulty level at any time if the current image isn't suitable.

Using the Description Practice

  1. Show the generated image to the child
  2. Ask them to describe what they see in the image
  3. Type their description into the chat input box
  4. Click "Submit" to receive AI feedback
  5. Share the feedback with the child
  6. Continue the conversation, encouraging the child to notice more details

Tip: Encouraging Description

Try open-ended prompts like "What do you see in this picture?", "Tell me about the colors you notice", or "What is happening in this image?"

Understanding the Progress Tracking

  • Details Checklist: Shows all the key elements in the image and whether they've been identified
  • Attempt Counter: Displays how many attempts have been used out of the allowed total
  • Progress Bar: Shows how close the child is to reaching the threshold for advancement
  • Session History: Records all past sessions for review and assessment

Tip: Interpreting Feedback

The AI feedback is designed to be encouraging while guiding the child toward noticing more details. Hints are provided when the child seems to be struggling with identifying important elements.

Saving and Exporting Data

  • Save All Session Images: Exports all generated images to a folder on your computer
  • Save Session Log: Creates a JSON file with complete session data for record-keeping or analysis

Tip: Regular Backups

Consider saving session data at the end of each learning session to build a comprehensive record of progress over time.

Troubleshooting

Common Issues

Image generation is taking too long
Try reducing the number of inference steps in the configuration or check your internet connection.
The system doesn't recognize details the child mentions
The AI tries to match concepts rather than exact words. If recognition is still an issue, try encouraging the child to be more specific or use different words to describe the same elements.
Error messages during image generation
Check that your API keys are valid and correctly entered in the .env file. If problems persist, try restarting the application.
